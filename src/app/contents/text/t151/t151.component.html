Antes de adentrarnos en el concepto de variable debes de conocer los datos,  y cuál es la función que desempeñan. 

La computadora se puede considerar como una unidad en la que se colocan  ciertos datos (entrada “input”), se procesan y se produce un resultado (salida  o información “output”). Los datos de entrada y los datos de salida pueden  ser, realmente, de cualquier tipo: texto, dibujos, sonido, imágenes, etc. 

Una colección de datos que se proporcionan a los algoritmos que se han de  ejecutar para encontrar una solución. 

Tipos de datos  

• Simples 

o Ocupan una sola casilla de memoria 

o Hace referencia a un único valor a la vez 

o Pueden ser enteros, caracteres, booleanos 

• Estructurados  

o Tienen varios componentes que pueden ser datos simples o estructurados 

o Hace referencia a un grupo de casillas de memoria 

o Pueden ser, arreglos, cadenas de caracteres, registros, conjuntos. 


1) Datos numéricos 

a. Enteros como: 128, 45637, 4, -1223, -12, etc. 

b. Reales como: 7.5, 37.4875, -1399.0987, -3.87, etc. 

2) Datos alfanuméricos  

a. Caracteres como: (a, b, c, …z), (0,1,2, …9), (#, $, *, °, %, /), debes de  tener en cuenta que un número que se declara carácter no se puede  utilizar para operaciones aritméticas. 

b. Cadenas de caracteres como: “abcde”, “mate$#*”, “22-2-6-49”,  “34abc”.

3) Datos lógicos 

a. Booleanos: son datos que sólo pueden tomar dos valores: verdadero  (true) o falso (false). 

Identificadores 

Los datos a procesar por una computadora ya sean simples o estructurados, deben almacenarse en casillas o celdas de memoria para su posterior  utilización. Estas casillas o celdas de memoria (constantes o variables) tienen  un nombre que permite su identificación. Llamaremos identificador al  nombre que se les da a las casillas de memoria. 

Regla para los identificadores 

o El primer carácter que forma un identificador debe ser una  letra (a, b, c, z).  

o Los demás caracteres pueden ser letras (a,b,c,...,z), dígitos  (0,1,2,...,9) o el símbolos especiales 

Los identificadores son las “Etiquetas”, y se recomienda que tengan un  nombre representativo de su tarea. 

Ejemplos 

o Suma 

o X7 

o aux 

o NUM 

o Num_2 

Constantes y variables 

Como hemos visto el computador sigue una serie de instrucciones. Pero estas  instrucciones tienen que operar sobre una serie de datos. El compilador típico solo procesa una instrucción a la vez, por lo que necesita espacios de  memorias donde guardar o depositar, a modo de cajones, por usar un símil  conocido, los diversos datos con los que trabaja. Aquí es donde entran en  juego las variables y constantes.

Las variables y constantes contienen los siguientes elementos:  

▪ Nombre: a través del cual el programa accede y manipular esa  posición de memoria. 

▪ Referencia del dato que va a contener 

▪ Tipo: determina el tipo de datos que contendrá la variable.  Puede ser entero, etc.  

Las constantes son datos que no cambian durante la ejecución de un  programa, mientras que las variables si pueden cambiar su valor. 

Ejemplos 

o num=12 (El identificador en num, el tipo es entero, y hace  referencia al valor 12) 

o nombre_2= ”Javier” (El identificador es nombre_2, el tipo es  cadena de caracteres y hace referencia a la cadena “Javier”). 

o div_/= 14.78 (El identificador es div_/, el tipo es real, y hace  referencia al valor 14.78).
