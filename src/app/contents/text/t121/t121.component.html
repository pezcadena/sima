Ahora que has analizado que es un problema y que para resolverlo  necesitamos realizar una serie de pasos para llegar a un resultado, es  momento de analizar una de las posibles técnicas que puedes usar para  resolver dichos problemas 

Las técnicas de solución concretas aplicables para resolver un problema  varían según el tipo de problema del que se trate y según la experiencia y  conocimientos que se tengan. 

Resolver problemas usando analogías, consiste en recordar problemas  similares, en los que las relaciones entre sus elementos sean concordantes  con las de nuestro problema. 

Al buscar problemas similares al nuestro, pretendemos buscar problemas  mas sencillos, de tal manera que podamos aplicar sus métodos o resultados  para de esta manera poder encontrar la solución de nuestro problema con  mayor facilidad. 

De manera general este tipo de técnica consiste en relacionar un problema o  una situación, con otra semejante que ya se ha resuelto anteriormente. Es  decir, identificar algún patrón común. 

Por ejemplo, es posible prever algunas características del resultado que sean  más o menos plausibles si se analiza un problema de fisiología humana a  partir de una analogía con la fisiología de la rana.  

Pero para aplicar adecuadamente una analogía, el investigador debe,  primero, familiarizarse con los conceptos básicos subyacentes en el problema  para definir los límites de la analogía y no forzarla a conclusiones erróneas.
